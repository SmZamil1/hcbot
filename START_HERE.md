## 📋 The 5 Critical Steps

No matter which path, these 5 things MUST happen:

1. **Edit config.php**
   - Add your database credentials from cPanel
   - Add your bot token
   - Add your admin ID

2. **Create Database in cPanel**
   - Create MySQL database
   - Get credentials
   - Paste them in config.php

3. **Upload Files to InfinityFree**
   - All 12 files to `/public_html/`
   - Keep the folder structure (bot/, api/, admin/)

4. **Import database.sql**
   - Use phpMyAdmin → Import
   - Select database.sql file
   - Click Go

5. **Set Telegram Webhook**
   - Copy URL from SETUP.md
   - Paste in browser
   - Verify success response
